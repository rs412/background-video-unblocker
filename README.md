# Background Video Unblocker
一款 Manifest V3 轻量浏览器扩展，拦截网页切屏检测，实现后台持续播放视频，兼容 Chrome / Edge

## 项目介绍
Background Video Unblocker 是无广告、无网络请求的本地浏览器插件，专门拦截网站用来监控页面状态的 `visibilitychange`、`blur`、`pagehide`、`mouseleave` 等检测事件。
插件会伪造页面永久可见、窗口持续聚焦，同时定时自动恢复被强制暂停的视频，切换标签、最小化窗口、鼠标移出浏览器时视频不会中断。

## 核心功能
1. 拦截网页检测切屏、失焦、页面隐藏的监听事件
2. 覆写页面可见性API，欺骗网站页面始终处于前台
3. 定时检测页面视频，自动续播被暂停的视频
4. 右上角弹窗一键开关全部功能，随时启用/禁用
5. 本地持久化存储配置，重启浏览器设置不丢失
6. 支持普通网页与 iframe 嵌套网课页面
7. 纯本地运行，不会上传任何数据，体积小巧

## 项目文件结构
Background-Video-Unblocker/
├─ icon.png 扩展图标
├─ manifest.json Manifest V3 扩展配置文件
├─ content.js 页面注入核心拦截脚本
├─ popup.html 插件设置弹窗页面
└─ popup.js 弹窗开关与本地存储逻辑

## 本地安装教程
1. 将仓库全部文件下载或克隆到本地同一个文件夹
2. 打开 Chrome / Edge，地址栏输入对应地址：
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
3. 开启页面右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选中项目文件夹
5. 浏览器右上角出现插件图标，点击即可打开功能开关

## 使用说明
1. 点击浏览器工具栏插件图标，弹出简易开关；
2. 打开开关代表启用全部拦截功能，关闭则插件不生效；
3. 设置自动保存在本地，关闭浏览器、重启电脑后配置保留；
4. 修改开关状态后，刷新网页即可立即生效。

## 重要须知
1. 本插件仅屏蔽前端 JavaScript 页面检测；
2. 如果平台存在后端计时、摄像头监控、客户端监考软件等校验手段，本插件无法绕过；
3. 请勿用于违规线上考试场景，请合理合规使用。
